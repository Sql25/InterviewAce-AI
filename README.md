# 🤖 InterviewAce AI

A Streamlit-based SQL Interview Practice App designed to help candidates prepare for data analyst and finance roles.

## 🚀 Features
- Section A: SQL MCQ Test (100 marks)
- Section B: Scenario-based questions
- Instant scoring and feedback
- Clean UI for interview simulation

- Streamlit

## 🎯 Use Case
Helps candidates practice real-world SQL and interview scenarios with structured evaluation.
